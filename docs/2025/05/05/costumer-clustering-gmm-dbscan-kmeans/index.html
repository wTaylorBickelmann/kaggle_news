Key Steps and Unusual Moves

1. Smart Missing-Value Treatment
• One missing “CREDIT_LIMIT” row is simply dropped.
• A custom imputation of NaNs in “MINIMUM_PAYMENTS” hinges on each user’s total payments:
– If payments = 0 → minimum payment set to 0
– If payments < average → minimum payment assumed equal to payments
– Otherwise → fill with the overall average payment
This conditional logic preserves subtle distinctions in payment behavior that a straight mean‐fill would obscure.
